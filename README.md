# CMS Carts and Seds

## Description
18F/CMS CARTS & SEDS redevelopment 

## Usage
1. Clone the repository
2. CD in to the directory
    1. `cd cms-carts-seds`
3. Install npm dependencies
    1. `cd react`
    2. `npm install` 
4. Start the container
    1. `cd ../` 
    2. `docker-compose up`
5. Visit site at http://localhost:3000
